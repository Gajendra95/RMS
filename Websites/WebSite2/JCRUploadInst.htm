Instructions to generate the JCR file
1.Login to WebofScience website-http://apps.webofknowledge.com
2.Click on the tab-Journal Citation Reports
3.Pull out the Journal Citation Report by selecting appropriate Year and Journal Category
4.Adjust the filtering criteria so that Number of results is within 500 records at a time and finalize the Report
5.Mark all the items and export the result by clicking on the tab-Marked List
6.The exported result is saved as a Text file (abc.txt) clicking on the tab-Save To file
7.Open the file in a Text Editor – Search/Replace ; to , and Save the file
8.Rename the file as abc.csv
9.Complete the upload by selecting the abc.csv using RMS tool
10.RMS tool gives the confirmation on Data update – Data is updated successfully.
